Discount Bootstrap

Features

  • Column-Based Responsive Grid

    expand_more

    A system of div classes to organize a page into columns and rows responsively, from span1 (1/12th of the page) to span12 (100% of the page).

  • Spacing System

    expand_more

    Separated out to be more customizable, organizes how the responsive grid items are spaced on both mobile and desktop.

  • Indentations for Mobile & Tablet

    expand_more

    "Indents" text on certian media queries so it isn't touching the sides in an unattractive way.

  • Accordion

    expand_more

    The item you're looking at! Allows for expandable information piece by piece. TO-DO: add option to make it so opening a different part of the accordion collapses the previously opened one.

  • Toggle

    expand_more

    A concept for a toggle that passes the element toggled and the ID of that element into a function that chooses which function to run. Can't decide if this is efficient or not. (TO-DO)

  • View/See More

    expand_more

    Similar to the accordion, but generally meant more for single pieces of information (like long paragraphs or series of images, shows more content when "view more" or a similar button is clicked.

  • Gallery

    expand_more

    Cycles through different divs, which can contain anything from text to images, with a recognizable navigation option at the bottom of several circles, each representing the divs that make up the gallery.

  • Anchor System

    expand_more

    System for adding anchor points and references to them.
